#Unit testing

##Unit testing with a browser

```sh
git checkout browser
open test.html
```

##Unit testing with multiple browser (karma)

```sh
git checkout karma
npm install
karma start
```

##Unit testing with node
```sh
git checkout node
npm install
karma start
```

##Unit testing with both
```sh
git checkout both
npm install
karma start
mocha
```

##Unit testing with both (grunt)
```sh
git checkout grunt
npm install
karma start
mocha
```
